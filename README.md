# MediAssist PWA

MediAssist is a simple Progressive Web App for managing patient visits.
It’s lightweight, works offline, and can be installed like a normal app on mobile or desktop.

## Features

* User-Friendly UI
* Works offline after first load
* Installable as a PWA
* Fast and lightweight
* No frameworks or dependencies

## Tech Stack

* HTML
* CSS
* Vanilla JavaScript
* Service Worker

## Offline Support

The app caches core files using a Service Worker.

## Install as App

* Open in Chrome or Edge
* Works offline after installation

## Notes

* Designed to stay lightweight and simple
* Current Version - v4.0
